
# ✈️Tiket Penerbangan 

Program ini adalah sebuah program sederhana dalam bahasa pemrograman C++. Program ini mengelola data tiket pesawat pada sebuah agen perjalanan.


## Spesifikasi Tiket

- Maskapai
- Kelas
- Tujuan
- Harga Per Tiket
- Jumlah Tiket Yang Dipesan
- Total Harga
